You can set the language direction in the `languages.md` config file.

- `rtl` - right to left
- `ltr` - left to right

```toml
[cn]
  title = "Zzo"
  languageName = "English"
  weight = 1
  languagedir = "rtl"

[ko]
  title = "Zzo"
  languageName = "한국어"
  weight = 2
  languagedir = "ltr"
```
