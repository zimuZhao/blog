1. Add this line of code to your params.toml file.

    ```
    ...
    custom_js = ["js/custom.js", ...]
    ...
    ```

    `config/_default/params.toml`

2. Add your file to assets folder. Filename must match with config params you set above.

    ```
    assets/js/custom.js
    ```
